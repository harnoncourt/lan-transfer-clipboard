# 项目概览

## 产品定位

LAN Transfer Clipboard 是一个局域网传输工具，用于在个人设备之间快速发送：

- 文件。
- 剪贴板文本。

目标用户是同时使用多台设备的人，例如：

- Mac 与 Windows 之间传文件。
- Mac 与 Android 之间传文本。
- iPhone 与电脑之间临时交换文件。
- 同一办公室或家庭网络内的设备快速互传。

## 核心价值

- 不需要云盘。
- 不需要登录账号。
- 不需要数据经过第三方服务器。
- 局域网内速度快，隐私边界更清晰。
- 一套 Flutter 客户端覆盖桌面和移动端。

## MVP 范围

当前 MVP 聚焦验证最关键的端到端链路：

- 发现设备。
- 选中设备。
- 发送剪贴板。
- 发送文件。
- 接收内容。
- 展示接收记录。

## 非目标

当前阶段暂不做：

- 云同步。
- 用户账号系统。
- 跨公网传输。
- 群发。
- 文件夹同步。
- 网盘式文件管理。
- 自动剪贴板双向同步。

这些能力可以作为后续产品方向，但不属于当前 MVP。

## 成功标准

短期成功标准：

- 两台同一局域网设备能互相发现。
- 能稳定发送小文件和文本。
- 发送失败时用户能看到错误。
- 平台权限配置清晰。

中期成功标准：

- 有可信设备配对。
- 传输内容加密。
- 有传输进度、取消和失败重试。
- 移动端体验符合系统限制。

长期成功标准：

- 跨平台发布包可安装。
- 普通用户无需理解网络细节即可使用。
- 局域网内传输稳定、可解释、可恢复。
